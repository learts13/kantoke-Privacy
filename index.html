Política de Privacidade

Versão: 1.0.13
Última atualização: 10 de maio de 2026
Aplicativo: Kantokê

1. Introdução

O Kantokê é um aplicativo desenvolvido para auxiliar usuários na gestão de acervos de músicas para videokê. Este documento descreve de forma transparente quais dados são coletados, em conformidade com a LGPD.

2. Dados Coletados

2.1 Dados Locais

Músicas favoritas e configurações são armazenadas exclusivamente no seu dispositivo via SQLite.

2.2 Autenticação Google

Utilizamos o Firebase Auth para login opcional. Coletamos apenas UID, e-mail, nome e foto de perfil para identificar sua conta.

3. Compartilhamento

Não vendemos os seus dados. Compartilhamos apenas o estritamente necessário com Google AdMob (anúncios) e Firebase (autenticação).
